My Projects

Some of the projects I've worked on. Some academic. Some professional.

Lending Stream

Designed and developed Lending Stream website, its features and functionality from 2010 to 2013.

Zebit

Planned and built software that supports various features of Zebit. Developed the Zebit API on top of which the Zebit Mobile Application is being built.

Page Relevance Prediction

Machine Learning - Built a linear regression model using MATLAB to predict the relevancy of search results. Used Microsoft's Learning To Rank dataset.

Classification of Handwritten Numerals

Machine Learning - Build two classifier models to classify between handwritten numerals 0-9 and compared results with Neural Network classifier

Wikipedia Indexer

Built a parser on structured Wikipedia documents. Tokenized the contents using numerous rules such as stemming, lemmatizing, etc. Created indexes on the data and built an index analyzer and optimized it.

Q&A System using Apache SOLR

Built a Question Answer system using data parsed and indexed from Wikipedia Infoboxes using Apache SOLR. The front-end provides an easy interface to interact with the index.
